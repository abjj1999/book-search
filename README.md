## How to use

```bash
# Clone this repository
$ git clone https://github.com/abjj1999/book-engine

# Install dependencies
$ npm install

# Run the app
$ npm run develop
```

## Built with 
- MERN (MongoDB, Express.js, React.js, Node.js)
- Apollo
- GraphQL
- BootStrap
